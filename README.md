A simple app that demonstrates the following concepts:

DAY 1

Lists names from a variable

Created a ListItem composable function to display each name inside a card with text

Used Card to create a card for each name

Used the Modifier function to control the size and padding of UI elements

Used the vector asset baseline_person_24 as a placeholder for a person's image

Used Row to arrange items horizontally

DAY 2

Used a data class to store and manage data

Added firstName, lastName, and age properties to the data class

Used the people.filter function to filter values based on age

DAY 3

Added a data class for items and an enum class for item types

Used if statements to perform different actions depending on the item type
